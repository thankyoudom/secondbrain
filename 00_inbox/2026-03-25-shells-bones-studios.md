# shells & bones studios

This is actually a strong instinct—you’re thinking like a studio founder, not just an artist. The key shift in this “next chapter” is:
you’re no longer optimizing for your output, you’re optimizing for a system that produces output.

Let’s map a realistic, grounded 1-year version of this.

?

? Year 1: “Micro-Studio” Phase (Post-Residency)

? Objective

### Turn early success (2 mangas) into
|  | • | A repeatable creative pipeline |
| --- | --- | --- |
|  | • | A small but serious team |
|  | • | 2–4 polished outputs (not 10 scattered ones) |
?

? Team Structure (Lean but Real)

You + 2–3 apprentices is actually the sweet spot.

Core Team (4 total max)
|  | • | You (Creative Director / Lead Artist) |
| --- | --- | --- |
|  | • | Vision, final polish, storytelling, style consistency |
|  | • | Apprentice 1 (Linework / Inking / Panels) |
|  | • | Apprentice 2 (Color / Shading / Backgrounds) |
|  | • | Optional Apprentice 3 (Flexible Role) |
|  | • | Could lean into animation, editing, or social content |
? Don’t go beyond 3 yet.
More people = management overhead > creative output at this stage.

? Output Strategy (This is where most people mess up)

Do NOT try to do everything.

|  | • | Manga = faster iteration + builds IP |
| --- | --- | --- |
|  | • | Film = proof of expansion + attention grabber |
