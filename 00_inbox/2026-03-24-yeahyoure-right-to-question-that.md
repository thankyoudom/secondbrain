# Yeah—you’re right to question that.

The Artist’s Way does technically include more than just Morning Pages + Artist Dates. I simplified earlier on purpose—but let’s zoom it back out so you see the full picture without overwhelming yourself.

?

What’s actually in the curriculum

### Each week has

1. The Core (non-negotiable)
|  | • | Morning Pages (daily) |
| --- | --- | --- |
|  | • | Artist Date (weekly) |
These are the engine. Everything else supports this.

?

2. Weekly Reading

Each week has a chapter with themes like:
|  | • | Week 1: Recovering a sense of safety |
| --- | --- | --- |
|  | • | Week 2: Identity |
|  | • | Week 3: Power |
|  | • | etc. |
### ? This is important, but
|  | • | You don’t need to “study” it |
| --- | --- | --- |
|  | • | Just read and let it sit |
?

3. Weekly Tasks (this is what you’re thinking of)

Each chapter gives a list of exercises like:
|  | • | Write out negative beliefs (“blurts”) |
| --- | --- | --- |
|  | • | List people who supported your creativity |
|  | • | Do small creative challenges |
|  | • | Track synchronicities |
|  | • | Declutter something |
|  | • | Write letters (you don’t send) |
These are where people burn out if they try to do all of them perfectly.

?

The truth about the tasks (based on real experiences)

### Most people
|  | • | Don’t complete every task |
| --- | --- | --- |
|  | • | Skip some weeks |
|  | • | Still get massive value |
### The author even implies

It’s not about perfection—it’s about engagement.

?

How to approach the “extra items” without burnout

### Here’s the move for you

Think of tasks as a menu, not a checklist

### Each week
|  | • | Pick 2–4 tasks max |
| --- | --- | --- |
|  | • | Choose the ones that hit something real |
### Not

“Let me complete everything”

### But

“Which of these actually matters to me right now?”

?

What this should look like in practice (your version)

### Weekly flow

1. Read the chapter (once)
- No notes needed unless something hits
2. Pick a few tasks
### Example
|  | • | 1 introspective (beliefs, fears) |
| --- | --- | --- |
|  | • | 1 action-based (declutter, try something) |
|  | • | 1 creative (music, idea, exploration) |
3. Keep Morning Pages + Artist Date steady

That’s it.

?

Where people like you go wrong (important)

You’re capable, structured, and ambitious—so your instinct is:

“Let me execute the full system properly”

### That’s exactly how you accidentally
|  | • | turn it into work |
| --- | --- | --- |
|  | • | lose the creative benefit |
|  | • | burn out or drop it |
?

The better mindset

### Treat it like

“I’m participating in this, not completing it.”

?

A simple rule to keep you aligned

### If at any point you feel
|  | • | pressure |
| --- | --- | --- |

| --- | --- | --- |
