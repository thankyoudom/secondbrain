# curriculum

Got it — I’ll treat this like building you a full, step-by-step self-study program that takes you from your current full-stack + Firebase + GCP + infra interest into a job-ready Cloud Engineer with the kind of skills and certifications that get noticed on LinkedIn and in interviews.

We’ll also include Terraform, ArgoCD, Kubernetes, and networking so you’re production-ready.

## Cloud Engineer Curriculum — Free-First Approach
(Estimated 4–6 months if done consistently)

PHASE 1 — Cloud Foundations & Networking Basics
Goal: Understand core cloud concepts, common architectures, and networking fundamentals.
1. Free Courses & Certs
Google Cloud Digital Leader (Free Training + Free Cert Voucher sometimes available)
Training: Google Cloud Digital Leader Learning Path (free)
Cert: Search “Google Cloud Free Digital Leader Voucher” — often offered for free.
Recognized on LinkedIn ?
AWS Cloud Practitioner Essentials (Free on AWS Skill Builder)
AWS Free Course
Even if you don't certify, this gives you multi-cloud awareness.
Microsoft Learn Cloud Fundamentals (Free, LinkedIn badge)
Microsoft Learn Cloud Skills
2. Networking Basics (must-have for cloud engineers)
Free Cisco Intro to Networking: Cisco Networking Basics — LinkedIn badge ?
### Study topics
TCP/IP, DNS, DHCP, HTTP/S
CIDR notation, subnets, routing tables
VPCs, firewalls, load balancers

PHASE 2 — Linux, Git, and DevOps Fundamentals
Goal: Comfortably work in terminal, automate tasks, and manage code/infrastructure.
1. Linux & Bash
Linux Essentials (Cisco Skills for All, free LinkedIn badge)
Linux Unhatched
Learn navigation, permissions, processes, SSH.
2. Git & GitHub
GitHub Skills (LinkedIn-ready badges for each mini-course)
Learn: branching, merging, pull requests, tags, version control best practices.
3. Containers & Kubernetes Basics
Kubernetes for Beginners (FreeCodeCamp, 4 hours)
YouTube course
Docker Essentials (IBM SkillsBuild, free cert)
IBM Docker Basics

PHASE 3 — Infrastructure as Code (Terraform)
Goal: Learn to provision cloud infrastructure in a repeatable way.
1. Terraform Basics
Free HashiCorp Terraform Associate Prep (Study without paying for cert)
HashiCorp Learn Terraform
### Topics to master
Providers, resources, variables, outputs
Remote state, workspaces
Terraform Cloud basics
2. Practical Projects
Deploy a simple GCP VM + VPC with Terraform
Deploy a Kubernetes cluster with Terraform

PHASE 4 — CI/CD & GitOps (ArgoCD)
Goal: Understand continuous delivery workflows and GitOps principles.
1. ArgoCD
Free Hands-on: ArgoCD Official Getting Started
### Learn
App-of-Apps pattern
Sync policies (automatic vs. manual)
Managing multiple environments
2. CI/CD Basics
Google Cloud Build + GitHub Actions (free tiers)
### Learn
Automated testing pipelines
Deployment triggers from Git pushes

PHASE 5 — Google Cloud Associate Cloud Engineer (ACE) Prep
Goal: Prepare for and pass the ACE — your only paid cert in this plan.
1. Official Resources
ACE Exam Guide
### Free training
Coursera ACE Prep — Free if audited
2. Practice Labs (Free)
Qwiklabs / Google Cloud Skills Boost — look for free monthly credits.

PHASE 6 — Resume Boost & Portfolio
Goal: Have visible proof of your skills.
1. Free Certs with LinkedIn Badges to Do Along the Way
Cisco Skills for All: Networking, Linux, Cybersecurity
IBM SkillsBuild: Docker, Cloud Essentials
Microsoft Learn: Azure Fundamentals badges
GitHub Skills: Actions, Pages, Codespaces
2. Portfolio Projects
Deploy a multi-tier web app on GCP with:
Load balancing
Managed database
CI/CD pipeline
Infrastructure via Terraform
ArgoCD for deployments
Document everything in GitHub repos.

PHASE 7 — Job Readiness
### Practice interview questions for
Cloud concepts (VPC, IAM, storage)
Networking (CIDR, routing)
Terraform basics
Kubernetes concepts
### Join free communities
Google Cloud Community
DevOps Discord
