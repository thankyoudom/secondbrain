# My Neovim Cheatsheet

## Modes
- `Esc` Ôćĺ Normal mode
- `i` Ôćĺ Insert mode
- `v` Ôćĺ Visual mode
- `:` Ôćĺ Command mode

## Save/Quit
- `:w` Ôćĺ Save
- `:q` Ôćĺ Quit
- `:wq` Ôćĺ Save & quit
- `:q!` Ôćĺ Quit without saving

## Movement
- `hjkl` Ôćĺ Left/Down/Up/Right
- `w/b` Ôćĺ Next/Previous word
- `0/$` Ôćĺ Start/End of line
- `gg/G` Ôćĺ First/Last line

## Editing
- `dd` Ôćĺ Delete line
- `yy` Ôćĺ Copy line
- `p` Ôćĺ Paste
- `u` Ôćĺ Undo

## My Custom
- `Space w` Ôćĺ Quick save
- `Space c` Ôćĺ LLM cleanup
- `Space f` Ôćĺ Find files
